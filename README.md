# rag7 - Ragamuffin

![Ragamuffin](https://via.placeholder.com/800x400/1a1a2e/16c2d5?text=Ragamuffin+Visual+Agent+Builder)

**Ragamuffin** is a visual agent building platform that combines LangFlow, FastAPI, and React to create a powerful development environment for AI agents.

## Quick Start

```bash
# Start all services
./start-dev.sh

# Stop all services
./stop-dev.sh
```

## Documentation

- [Monorepo Overview](README_MONOREPO.md) - Complete architecture and features
- [Run Commands](RUN_COMMANDS.md) - Detailed setup and development instructions

## Services

- **LangFlow UI**: http://localhost:7860 - Visual flow builder
- **Backend API**: http://localhost:8000 - Flow management API
- **Frontend**: http://localhost:8080 - Main application interface
